No more same looking out-of-the-box websites.

You have to build your theme with variables, mixins and their css classes - and that is exactly what KSFace stands for.

Focused on the main things

Less is more.

Contains only the basic components you really need - like a grid system, responsive classes, minimalist color palette and typography. Everything else can be designed with CSS classes, variables and mixins.

Works perfect with Sketch

A paradise for designers.

Thanks to KSFace's minimalist philosophy, it fits perfect with Sketch. Simply design your cards, buttons and other website elements with Sketch and export the CSS code into your dynamic KSFace stylesheet mixins.
